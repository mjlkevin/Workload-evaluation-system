# Workload-evaluation-system

工作量评估系统（轻量计算导出型）代码骨架。

## Repo layout

- `apps/web`：Vue 3 + Vite 前端骨架
- `apps/api`：TypeScript + Express API 骨架
- `config/templates`：模板配置示例
- `config/rules`：规则配置示例

## Quick start

```bash
npm install
npm run dev:web
npm run dev:api
```
