# 需求管理

本目录存放原始需求与过程材料；**已定稿的产品与交互口径**同步体现在：

- `02_产品设计/`（视觉规范、预置模式等）
- `00_项目治理/里程碑与计划/项目进展总结与后续规划.md`（当前能力边界与后续规划）
- `03_技术设计/系统演进/实现与文档对齐说明.md`（实现事实）

## 原始需求索引（示例）

| 类型 | 路径说明 |
|------|-----------|
| 资源人天模板 | `原始需求/项目资源人天及成本表模板.xlsx` |
| 开发评估原型 | `原始需求/开发评估页-原型.xlsx` |
| Excel 解析样例 | `原始需求/excel_parse_report.txt` 等 |

新增需求请按日期或主题放入 `原始需求/`，并在里程碑或进展文档中追加一条变更记录。
